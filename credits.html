Créditos del Proyecto

Desarrollo y Diseño

Desarrollado como proyecto final para el Grado Superior de Desarrollo de Aplicaciones Web (2º DAW). Enfocado en la gestión integral de refugios felinos.

Tecnologías Utilizadas

  • Backend: PHP 8.x con arquitectura REST API.
  • Base de Datos: MySQL / MariaDB.
  • Frontend: HTML5, CSS3 (Vanilla + Bootstrap 5.3) y JavaScript Moderno (ES6+).
  • Autenticación: Sistema de sesiones PHP y hashing BCRYPT.
